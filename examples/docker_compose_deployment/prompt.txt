CORE SMIA CONTEXT (WHY/HOW)
SMIA links CSS concepts (capabilities/skills) to AAS elements through semantic identification (semanticId IRIs) in OWL ontologies. SMIA consumes an AASX + ontology + configuration, then generates/runs an industrial agent (SPADE/XMPP) able to coordinate and execute skills by invoking asset interfaces described in the AAS (via AssetInterfacesDescription with forms/endpoints).
